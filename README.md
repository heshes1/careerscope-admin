# CareerScope Admin

CareerScope Admin is a React-based admin dashboard that manages professional experience entries, tracks skill analytics, and generates CV-ready summaries. It demonstrates authentication flows, CRUD operations, audit logging, and a clean blue/white admin UI.

## Key Features

- Demo login + protected routes
- CRUD table for experiences (search/filter/sort)
- Row selection + export selected as JSON
- Audit log for create/update/delete
- Skill analytics (top tags + category totals)
- Toast notifications (Saved/Deleted/Copied/Exported)
- localStorage persistence (no backend)

## Tech Stack

- React + Vite
- React Router
- Context API (toasts/auth)
- localStorage
- Vanilla CSS

## Usage Walkthrough

1. Login with the demo credentials.
2. Add a new experience entry from the admin form.
3. Select one or more rows and export selected records as JSON.
4. Review create/update/delete history in the audit log.
5. Open analytics to inspect top tags and category totals.
6. Copy the generated CV summary for reuse.
