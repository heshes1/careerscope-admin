# CareerScope Admin

CareerScope Admin is a React + Vite dashboard for managing professional experience entries, audit logs, and basic analytics.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Lint

```bash
npm run lint
```

## Format

```bash
npm run format
```
